**版本**: 1.0  
**日期**: 2026-04-10  
**状态**: MVP 开发中

## 1. 产品概述

### 1.1 产品定位

Saga Invoice 是一个面向国际电商卖家的免费在线发票生成工具，帮助用户在 30 秒内完成专业发票的创建和导出。

### 1.2 目标用户

| 用户类型 | 描述 | 核心需求 |
|----------|------|----------|
| Amazon 卖家 | 在 Amazon 平台销售商品 | 快速开具销售发票 |
| Shopify 卖家 | 独立站电商运营 | 品牌化发票模板 |
| Etsy 卖家 | 手工艺品卖家 | 简单易懂的操作 |
| 跨境电商 | 多平台运营 | 多货币、多语言支持 |

### 1.3 核心价值主张

> 30 秒内完成一张专业发票的创建和发送

### 1.4 竞品对比

| 产品 | 价格 | 是否需要注册 | 核心优势 | 我们的差异化 |
|------|------|-------------|----------|-------------|
| InvoiceCat | 付费 | 是 | 功能丰富 | 免费、无需注册 |
| Zoho Invoice | 付费 | 是 | 企业级功能 | 轻量、快速 |
| Wave | 免费 + 付费 | 是 | 完整财务套件 | 专注发票单一场景 |

| 优先级 | 功能模块 | 状态 | 验收标准 |
|--------|----------|------|----------|
| **P0** | 发票编辑器 | 进行中 | 可创建并保存发票 |
| **P0** | 本地存储 | 进行中 | 刷新后数据不丢失 |

| **P1** | CSV 导出 | 待开发 | 导出可打开的 CSV 文件 |
| **P1** | 客户管理 | 待开发 | 可保存和复用客户信息 |
| **P1** | 商品/服务库 | 待开发 | 可保存和复用商品条目 |

| **P3** | 云端同步 | V2 | 多设备数据同步 |

### 2.2 功能规格

#### F1 - 发票编辑器

**用户故事**: 作为卖家，我想要一个直观的编辑界面，以便快速填写发票信息。

**需求详情**:
- [ ] 三栏式布局：导航 + 编辑 + 预览
- [ ] 实时预览：输入内容即时显示
- [ ] 自动计算：小计、税额、总计自动计算
- [ ] 行项目：支持添加/删除/编辑多行商品或服务
- [ ] 税率配置：可自定义税率百分比

**验收标准**:
- 新用户无需指导即可在 1 分钟内完成第一张发票
- 所有金额计算精确到分
- 预览与最终导出效果一致

#### F2 - 本地存储

**用户故事**: 作为卖家，我希望我的输入自动保存，以便意外关闭后不会丢失数据。

**需求详情**:
- [ ] 自动保存：每次输入后自动保存到 localStorage
- [ ] 数据持久化：关闭浏览器后数据保留
- [ ] 恢复功能：重新打开时恢复上次编辑的内容
- [ ] 首次提示：告知用户数据存储位置和风险

**验收标准**:
- 刷新页面后数据完整恢复
- 关闭并重新打开浏览器后数据存在
- 清除浏览器缓存会丢失数据（预期行为）

#### F3 - PDF 导出

**用户故事**: 作为卖家，我需要下载 PDF 格式的发票，以便发送给客户。

**需求详情**:
- [ ] 一键导出：点击按钮即可下载 PDF
- [ ] 格式保真：PDF 与预览效果一致
- [ ] 专业排版：符合国际发票标准格式
- [ ] 文件命名：自动使用发票编号命名

**验收标准**:
- 导出的 PDF 可在常见阅读器中打开
- 金额、日期等关键信息清晰可读
- 排版在不同尺寸纸张上正常显示

#### F4 - CSV 导出

**用户故事**: 作为卖家，我需要 CSV 格式用于报税和财务记录。

**需求详情**:
- [ ] 导出所有行项目为 CSV
- [ ] 包含完整的发票元数据
- [ ] 支持批量导出

**验收标准**:
- CSV 可在 Excel/Google Sheets 中打开
- 中文字符无乱码
- 金额格式正确

#### F5 - 客户管理

**用户故事**: 作为卖家，我想要保存常用客户信息，以便快速复用。

**需求详情**:
- [ ] 添加新客户
- [ ] 编辑客户信息
- [ ] 删除客户
- [ ] 选择客户自动填充

**验收标准**:
- 客户信息保存在本地
- 开票时可快速选择客户
- 支持搜索客户

#### F6 - 商品/服务库

**用户故事**: 作为卖家，我想要保存常用商品或服务，以便快速添加到发票。

**需求详情**:
- [ ] 添加新商品/服务
- [ ] 编辑商品/服务信息
- [ ] 删除商品/服务
- [ ] 选择商品自动填充描述和单价

**验收标准**:
- 商品信息保存在本地
- 开票时可快速选择商品
- 支持搜索商品

| 指标 | 目标值 | 测量方式 |
|------|--------|----------|
| 首屏加载时间 | < 1.5s | Lighthouse |
| 最大内容绘制 | < 2.5s | Lighthouse |

### 3.3 国际化

| 语言 | 优先级 |
|------|--------|
| English | P0 (默认) |
