## PDF 导出问题

### 问题：PDF 导出失败/空白

**症状**: 点击下载 PDF 按钮后无反应，或下载的文件为空白。

**可能原因与解决方案**:

| 原因 | 解决方案 |
|------|----------|
| 页面未完全加载 | 等待页面加载完成后再试 |
| html2canvas 加载失败 | 检查网络连接，刷新页面 |
| 浏览器不支持 | 更换为 Chrome/Firefox/Edge |
| 内存不足 | 关闭其他标签页后重试 |

**调试步骤**:
```javascript
// 打开浏览器控制台 (F12),检查是否有错误信息
// 常见错误:
// - "html2canvas is not defined" → 资源加载失败
// - "Permission denied" → 跨域问题
```

---

### 问题：PDF 文字模糊

**症状**: 导出的 PDF 文字不清晰、有锯齿。

**解决方案**:
1. 确保浏览器缩放为 100%
2. 使用更高分辨率的屏幕
3. 尝试导出后放大查看（PDF 阅读器缩放）

**技术原因**: html2canvas 基于 Canvas 渲染，分辨率受屏幕 DPI 限制。

---

### 问题：PDF 中文乱码/缺失

**症状**: 中文字符显示为方框或乱码。

**解决方案**:
1. 确保系统安装中文字体
2. 切换语言为 English 测试是否为字体问题
3. 刷新页面后重试

**临时方案**: 使用浏览器打印功能 → 另存为 PDF

## 数据丢失问题

### 问题：刷新后数据消失

**症状**: 填写发票后刷新页面，数据为空。

**排查步骤**:

1. **检查是否为同一浏览器**
   - localStorage 是浏览器隔离的
   - Chrome 的数据在 Firefox 不可见

2. **检查是否清除了缓存**
   - 清除浏览器数据会删除 localStorage

3. **检查控制台错误**
   ```javascript
   // F12 打开控制台，查看是否有 Storage 相关错误
   // "QuotaExceededError" → 存储空间已满
   // "SecurityError" → 隐私模式限制
   ```

**解决方案**:
- 隐私浏览模式（无痕模式）会限制 localStorage
- 使用正常浏览模式

---

### 问题：只能保存少量数据

**症状**: 填写较多行项目后无法保存。

**原因**: localStorage 有配额限制（通常 5-10MB）。

**解决方案**:
1. 减少行项目数量
2. 定期导出并删除旧发票
3. 清理浏览器缓存释放空间

## 语言切换问题

### 问题：切换语言后部分文字未翻译

**症状**: 点击语言切换后，部分 UI 元素仍为原语言。

**解决方案**:
1. 刷新页面
2. 清除缓存后重试

**已知限制**: 
- 某些动态加载的内容可能需要刷新才能完全翻译

---

### 问题：语言设置无法保存

**症状**: 切换语言后刷新页面，恢复为默认语言。

**排查**:
```javascript
// 检查 localStorage 中是否有语言设置
localStorage.getItem('sagainvo:settings')
```

**解决方案**:
1. 确保浏览器允许 localStorage
2. 检查是否有浏览器扩展阻止存储
3. 尝试其他浏览器

---

## 模板应用问题

### 问题：应用模板后数据异常

**症状**: 点击应用模板后，发票内容不符合预期。

**排查步骤**:
1. 确认是否点击了确认对话框中的"确认"
2. 检查模板是否兼容当前版本

**解决方案**:
- 应用模板会**覆盖当前数据**，请先导出备份

---

### 问题：模板预览与实际不符

**症状**: 模板预览效果与应用后不一致。

**可能原因**:
- 屏幕分辨率差异
- 浏览器缩放设置不同
- 字体加载延迟

**解决方案**:
1. 确保预览和导出时缩放一致
2. 等待字体完全加载

---

## 图片上传问题

### 问题：上传 Logo/背景图失败

**症状**: 点击图片上传按钮后无反应或报错。

**排查步骤**:

1. **检查文件格式**
   - 支持的格式：PNG, JPG, JPEG, SVG
   - 不支持：GIF, WebP, BMP

2. **检查文件大小**
   - 最大允许：5MB
   - 建议：小于 500KB

3. **检查控制台错误**
   ```javascript
   // F12 打开控制台查看错误
   // "File too large" → 文件超过限制
   // "Invalid format" → 格式不支持
   ```

**解决方案**:
1. 压缩图片后重试
2. 转换为 PNG 格式
3. 使用更小的图片

---

### 问题：图片显示模糊

**症状**: 上传的 Logo 或背景图显示模糊。

**原因**: 图片分辨率不足或被压缩。

**解决方案**:
1. 使用高分辨率原图（建议 300 DPI）
2. PNG 格式优先
3. 避免过度压缩

## 性能问题

### 问题：页面加载缓慢

**症状**: 打开页面后长时间加载。

**可能原因**:
- 网络速度慢
- 浏览器缓存过多
- 设备性能不足

**解决方案**:
1. 检查网络连接
2. 清除浏览器缓存
3. 关闭其他标签页释放内存
4. 尝试其他浏览器

---

### 问题：输入响应延迟

**症状**: 输入内容后预览区更新缓慢。

**排查**:
```javascript
// 打开控制台查看是否有性能警告
// 检查 CPU 使用率是否过高
```

**解决方案**:
1. 减少行项目数量
2. 刷新页面
3. 关闭其他应用程序

## 移动端问题

### 问题：移动端布局错乱

**症状**: 在手机或平板上页面布局异常。

**已知限制**:
- 移动端为响应式简化布局
- 某些功能在移动端不可用

**解决方案**:
1. 使用横屏模式
2. 缩放至合适大小
3. 推荐使用桌面浏览器
